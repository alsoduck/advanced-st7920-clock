# Advaned ST7920 Clock
쓸만한 ST7920 시계

# 요구 사항
* u8g2
* RTClib

* ST7920 128x64 디스플레이
* DS3231 RTC

# 기능
* 현재 날짜와 요일 표시
* 오전/오후 표시
* 초시계
* 멀쩡한 UI 디자인
* 하루의 퍼센트 표시

# 연결 방법
## ST7920 LCD
|핀 번호|핀 이름|연결|
|--|--|--|
 | 1 | GND| GND|
 | 2 | VCC |5V|
 | 4 | RS | CS(SPI) |
 | 5 | R/W |  MOSI |
 | 6 | E | SCK |
 | 15 | PSB | 0(SPI 모드) |
 | 19 | BLA | 적절한 저항을 거쳐서 5V |
 | 20 | BLK | GND |
 
## DS3231 RTC
|핀 이름|연결|
|--|--|
| GND| GND|
| VCC | 5V |
| SCL | SCL |
| SDA | SDA |
